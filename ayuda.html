Inicio

La página de inicio ofrece una visión general del proyecto MotoGP-Desktop, incluyendo las últimas noticias, imágenes destacadas de circuitos y enlaces rápidos a todas las secciones principales. Desde aquí puedes acceder directamente a la información sobre pilotos, circuitos, clasificaciones, meteorología y juegos interactivos mediante el menú de navegación superior.

Los contenidos de la página principal se actualizan automáticamente y están optimizados para adaptarse a dispositivos móviles y pantallas de diferentes tamaños.

Piloto

Esta sección proporciona información detallada de los pilotos de MotoGP. Se incluyen datos personales como nombre, edad, altura y peso, así como el equipo actual del piloto y su historial deportivo completo.

Además, puedes consultar los resultados de la temporada actual y acceder a material multimedia relacionado con el piloto, como imágenes, vídeos y audios. También se proporcionan enlaces a recursos externos oficiales como la página de MotoGP o la biografía del piloto en Wikipedia.

Circuito

Aquí encontrarás información sobre los circuitos del campeonato de MotoGP. Para cada circuito se muestran detalles como longitud, número de curvas, ubicación y altimetría.

La sección permite cargar archivos HTML con información adicional de los circuitos, visualizar gráficos SVG de altimetría y mapas interactivos mediante archivos KML. Todos los elementos son interactivos y adaptables a diferentes tamaños de pantalla.

La altimetría y mapas se actualizan automáticamente cuando se cargan archivos compatibles, mostrando el recorrido completo del circuito y puntos de interés destacados.

Meteorología

Esta sección ofrece información meteorológica actualizada para los circuitos, incluyendo temperatura, velocidad del viento, humedad y probabilidad de lluvia.

Clasificaciones

La sección de clasificaciones permite consultar el ranking actualizado de pilotos tras la carrera y el último ganador de la misma.

Juegos

En esta sección se incluyen juegos interactivos relacionados con MotoGP, diseñados para mejorar la experiencia del usuario.

Juego de cartas de memoria

Un juego clásico donde el usuario debe emparejar cartas relacionadas con pilotos, motos o circuitos. Permite mejorar la memoria visual y el reconocimiento de elementos del campeonato.

Cronómetro

Simula un cronómetro para que puedas medir tiempos de vuelta. Puedes iniciar, pausar y reiniciar los tiempos, siendo útil para practicar estrategias de carrera o comparar tiempos entre distintos circuitos.

Test de usabilidad

Un pequeño test interactivo para evaluar la comprensión del usuario sobre la plataforma. Incluye preguntas sobre navegación y contenido.

Configuración de la base de datos

Permite al administrador modificar parámetros de la base de datos para vaciar, reiniar o exportar lo datos obtenidos de la encuesta.

Test del Cronómetro PHP

Esta herramienta comprueba el correcto funcionamiento del cronómetro con un interfaz de usuario sencilla.

Accesibilidad y compatibilidad

Todas las secciones han sido diseñadas siguiendo los estándares de accesibilidad WCAG 2.0, garantizando compatibilidad con lectores de pantalla, navegación mediante teclado y adaptabilidad a dispositivos móviles. El proyecto está optimizado para HTML5 y CSS3 válidos, sin errores ni advertencias en los validadores oficiales.
